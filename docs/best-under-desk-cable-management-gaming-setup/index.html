    Installation Tips for Gaming Setups

    Before You Install

    • Measure your desk width — don't guess on sizing
    • Count your cables — power strips, USB hubs, charging cables, etc.
    • Plan your routing — where do cables enter and exit the tray?
    • Consider future additions — leave room for new peripherals

    Installation Best Practices

    1. Install the tray first before routing any cables
    2. Use velcro ties to bundle cables before placing in tray
    3. Route power separately from data cables when possible
    4. Leave slack for devices that move (keyboard, controllers)
    5. Test desk movement if you have a standing desk

    Common Mistakes to Avoid

    • Over-stuffing the tray — cables need breathing room
    • Mounting too low — should clear your legs when seated
    • Ignoring cable weight — heavy power strips need support
    • Forgetting about access — you'll need to add cables later
