    Step-by-Step Setup Guide

    Phase 1: Hardware Installation (15 minutes)

    Step 1: Connect Sync Box to Gaming PC

    1. Power off your gaming PC and monitor
    2. Connect HDMI cable from PC graphics card to Sync Box HDMI Input 1
    3. Connect Sync Box HDMI Output to your gaming monitor
    4. Connect Sync Box power adapter and Ethernet cable to router
    5. Power on devices in order: Sync Box → Monitor → PC

    Step 2: Physical Light Placement

    • Monitor backlighting: Mount Hue Play bars vertically behind monitor edges
    • Ambient lighting: Install Lightstrips along desk back edge and ceiling
    • Zone mapping: Place lights to match screen quadrants (top-left, top-right, bottom-left, bottom-right)

    Pro tip: Leave 6 inches between lights and wall for proper color diffusion

    Phase 2: Software Configuration (10 minutes)

    Step 3: Download Hue Sync Box App

    1. Install Philips Hue Sync mobile app (iOS/Android)
    2. Ensure phone is on same Wi-Fi network as Sync Box
    3. App will auto-detect your Sync Box during setup

    Step 4: Create Entertainment Area

    1. Open Hue app → Settings → Entertainment areas
    2. Select Create area → Choose your gaming room lights
    3. Map each light to screen position using the virtual grid
    4. Name the area "Gaming Setup" for easy identification

    Step 5: Optimize Gaming Settings

    Sync Box Settings:
- Brightness: 75-85% (prevents eye strain)
- Intensity: High (for action games) or Medium (for RPGs)
- Speed: Fast (for competitive games) or Medium (for cinematic games)
- Entertainment area: Gaming Setup

    Phase 3: Advanced Gaming Optimization

    Step 6: Game-Specific Profiles

    For FPS Games (Call of Duty, Valorant):

    • Intensity: High
    • Speed: Fast
    • Brightness: 60% (reduces distraction)
    • Focus: Center screen only

    For RPG/Adventure Games (Witcher, Cyberpunk):

    • Intensity: Maximum
    • Speed: Medium
    • Brightness: 85%
    • Focus: Full room immersion

    For Horror Games (Resident Evil, Phasmophobia):

    • Intensity: High
    • Speed: Slow (for building tension)
    • Brightness: 40%
    • Focus: Perimeter lights only

    Step 7: Multiple Input Configuration

    1. Input 1: Gaming PC (primary)
    2. Input 2: PlayStation 5 / Xbox Series X
    3. Input 3: Nintendo Switch
    4. Input 4: Streaming device (Apple TV, Shield TV)

    Auto-switching tip: Enable CEC control for automatic input switching when you turn on different gaming devices.

    Troubleshooting Common Gaming Issues

    ❌ "No signal" or Black Screen

    Cause: HDMI handshake issue with gaming PC
    Fix:

    1. Check HDMI cable quality (use included or certified 18Gbps cable)
    2. Update graphics card drivers
    3. Set Windows display to 4K 60Hz (not higher refresh rates)
    4. Try different Sync Box HDMI input port

    ❌ Lights Not Syncing to Games

    Cause: Entertainment area not properly configured
    Fix:

    1. Verify all lights show up in Entertainment areas
    2. Re-run room mapping in Hue app
    3. Check that Sync Box input matches active gaming device
    4. Restart Hue Bridge (unplug 30 seconds, plug back in)

    ❌ Sync Lag/Delay During Gaming

    Cause: Network congestion or Wi-Fi interference
    Fix:

    1. Connect Sync Box to router via Ethernet cable
    2. Restart router and Hue Bridge
    3. Change router Wi-Fi channel (use WiFi Analyzer app)
    4. Move Hue Bridge closer to Sync Box (within 10 feet)

    ❌ Colors Look Wrong/Washed Out

    Cause: Monitor color space mismatch
    Fix:

    1. Set monitor to sRGB color space
    2. Adjust Sync Box brightness to 75-80%
    3. Use Natural color setting instead of Vivid
    4. Calibrate monitor using Windows Display settings
