    Step-by-Step Camera Setup (Govee Example)

    Time needed: 20 minutes

    Step 1: Physical Installation

    1. Mount LED strip around back/sides of your monitor
    2. Position camera on top-center of monitor (included magnetic mount)
    3. Connect power adapter and control box
    4. Download Govee Home app

    Step 2: App Configuration

    1. Add device in Govee Home app
    2. Connect to your 2.4GHz WiFi network
    3. Run camera calibration (follows colored squares on screen)
    4. Test sync with a colorful YouTube video

    Step 3: Gaming Optimization

    1. Set brightness to 70-80% for comfortable gaming
    2. Enable "Gaming Mode" to reduce unnecessary color changes
    3. Create profiles for different game genres:
      • FPS: Subtle, non-distracting
      • RPG: Full immersion, high saturation
      • Racing: High speed, reactive colors

    Pro Tips:

    • Position camera to see 80% of your screen (not 100%)
    • Use "Screen Color" mode, not "Video" mode for gaming
    • Test sync delay with fast-paced games and adjust sensitivity

    Step 1: Hardware Setup

    1. Install Govee DreamView G1 LED strip around monitor

    3. Test basic functions in Govee Home app

    Step 2: Software Installation

    1. Download Govee Desktop from official website
    2. Install and launch on your gaming PC
    3. Sign in with your Govee account
    4. Detect and connect your DreamView strips

    Step 3: Gaming Configuration

    1. Capture Region: Select your primary gaming monitor
    2. Performance Mode: Enable for reduced CPU usage during gaming
    3. Game Profiles: Create custom profiles:
      • Competitive FPS: Low brightness, minimal reaction
      • Single-player RPG: High saturation, full immersion

    • Reduce brightness to 30-40%

    • Set brightness to 60-70% (looks good on camera)
    • Enable "Smooth Transitions" to reduce jarring changes
    • Test with OBS to ensure no interference

    Gaming Performance Impact

    Camera-based sync: Zero performance impact (no PC resources used)
    Software sync: 1-3% CPU usage, 50-100MB RAM
    Native integration: Negligible impact (<1% CPU)
    Audio-reactive: Zero performance impact

    Optimization for Competitive Gaming

    For FPS games where every frame counts:

    1. Use camera-based sync (Govee Immersion) - zero performance impact
    2. Reduce light intensity to 30-40% to minimize distraction
    3. Disable rapid color changes that could break concentration
    4. Use "Ambient" mode instead of reactive sync during competitive matches
    5. Create "Competitive" profile with minimal lighting effects

    Troubleshooting Common Sync Issues

    Problem: Sync lag/delay

    • Solution: Use camera-based sync instead of software
    • Check: WiFi signal strength to LED strips
    • Adjust: Software capture refresh rate (lower = less lag)

    Problem: Colors don't match screen accurately

    • Solution: Calibrate camera positioning and white balance
    • Check: Monitor color settings and brightness
    • Adjust: LED strip color temperature and saturation

    Problem: Lights flicker during gaming

    • Solution: Enable "Gaming Mode" to reduce sensitivity
    • Check: PC performance (close unnecessary apps)
    • Adjust: Software sync refresh rate

    Problem: Setup doesn't work with HDR games

    • Solution: Switch to SDR for gaming, or use camera-based sync
    • Check: Windows HDR settings and game HDR options
    • Alternative: Use audio-reactive sync for HDR content

    Advanced Setup: Multi-Zone Sync

    For ultimate immersion: Set up different lighting zones that react to different screen regions.

    Equipment needed:

    • Multiple LED strips or smart light panels
    • Advanced software (Prismatik, Razer Synapse 3)
    • High-performance PC (for processing multiple zones)

    Zone setup example:

    • Top strip: Reacts to sky/ceiling colors in games
    • Side strips: React to left/right screen edges
    • Bottom strip: Reacts to ground/floor colors
    • Bias light: Static white for eye strain reduction
