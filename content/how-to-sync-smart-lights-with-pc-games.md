## Troubleshooting Common Sync Issues

**Problem: Sync lag/delay**
- **Solution:** Use camera-based sync instead of software
- **Check:** WiFi signal strength to LED strips
- **Adjust:** Software capture refresh rate (lower = less lag)

**Problem: Colors don't match screen accurately**
- **Solution:** Calibrate camera positioning and white balance
- **Check:** Monitor color settings and brightness
- **Adjust:** LED strip color temperature and saturation

**Problem: Lights flicker during gaming**
- **Solution:** Enable "Gaming Mode" to reduce sensitivity
- **Check:** PC performance (close unnecessary apps)
- **Adjust:** Software sync refresh rate

**Problem: Setup doesn't work with HDR games**
- **Solution:** Switch to SDR for gaming, or use camera-based sync
- **Check:** Windows HDR settings and game HDR options
- **Alternative:** Use audio-reactive sync for HDR content
